<h1>GeziRehberi</h1>

  <p>
      This project is a simple <strong>single-page HTML website</strong> created to practice
      basic HTML structure and list usage.
  </p>

  <h2><mark>Project Description</mark></h2>

  <p>
      The website is about <strong>Italy</strong>. It provides a short travel guide for people        who want to visit Italy, including places to see and restaurants to visit.
  </p>

   <h2><mark>Technologies Used</mark></h2>

  <ol>
        <li>HTML5</li>
   </ol>

  <h2><mark>HTML Tags Used</mark></h2>

  <p>The following HTML tags were used in this project:</p>
    <ol>
        <li><strong>&lt;ol&gt;</strong> – To create ordered lists</li>
        <li><strong>&lt;br&gt;</strong> – To separate content and improve readability</li>
        <li><strong>&lt;mark&gt;</strong> – To highlight headings</li>
        <li><strong>&lt;strong&gt;</strong> – To emphasize important words</li>
    </ol>
    <h2><mark>Page Structure</mark></h2>
    <p>
        The entire page consists of different lists created using the
        <strong>&lt;ol&gt;</strong> tag. Since the project is designed as a single page,
        line breaks were frequently used to avoid confusion.
    </p>

  <h2><mark>Purpose</mark></h2>
   <p>
        The main purpose of this project is to practice basic HTML tags,
        list structures, and text formatting.
    </p>
